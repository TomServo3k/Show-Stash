# Contributing

Thanks for your interest in Show Stash.

## Pull Requests

Before opening a pull request:

1. Keep changes focused.
2. Describe the user-facing behavior.
3. Update `README.md` when behavior, setup, controls, configuration, or project structure changes.
4. Bump `minor_version` in `manifest` for each project change.
5. Avoid committing private API keys, local device credentials, Firebase secrets, or Roku packaging credentials.
6. Test on a Roku device when the change affects channel behavior.

## Development Notes

Show Stash is a Roku SDK channel using BrightScript and SceneGraph XML. Runtime files are `manifest`, `source/`, `components/`, and `images/`.

# Support

For Show Stash support, contact Pearl Lane, LLC.

- Website: https://PearlLane.com/ShowStash
- Email: hello@PearlLane.com

When requesting support, include:

- Your Roku device model.
- Roku OS version.
- Show Stash version.
- A short description of what happened.
- Whether the issue involves TMDb lookup, Firebase household sync, or launching a streaming service.

Do not send private API keys, passwords, payment information, or sensitive account details in support requests.
